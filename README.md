## 功能

- [x] 紀錄出行場所
- [x] 紀錄的士號碼
- [x] 自動離開
- [x] 查看/刪除訪問出行記錄
- [x] 加密出行紀錄
- [x] 隱私模式(不儲存出行紀錄)
- [x] 生成二維碼
- [x] i18n
- [ ] 查看確診個案

## 開發者專區

### Local testing

```bash
npm install
npm start
```

### 自己 Host

```bash
npm install
npm run build
```

將/build 放入自己 web server 就 okay
